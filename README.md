# PortScannerTCP

#### INSTALLATION:

$ git clone https://github.com/samuelemaestri/PortScannerTCP


#### HOW TO USE:

$ cd PortScannerTCP

$ ./PortScannerTCP.sh [url] [port/ports]

#### EXAMPLE:

$ ./PortScannerTCP.sh google.com 80,443,8080

$ ./PortScannerTCP.sh google.com 80.443.8080

$ bash PortScannerTCP.sh google.com 1-200

$ sh PortScannerTCP.sh google 443
